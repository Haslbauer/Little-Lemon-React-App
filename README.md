# Little Lemon React App

## Description
This project is a small restaurant website built with React.

It was created as part of the Meta Front-End Developer course on Coursera. The goal was to practice working with components, props, state, and a clean project structure.

## Features
- Reusable React components
- Menu section
- Basic filtering functionality
- Responsive layout
- Work in progress

## Tech Stack
- React
- JavaScript
- CSS
- Vite

## What I learned
- Structuring a React project
- Working with components and props
- Managing basic state
- Building reusable UI elements

## Status
Work in progress. Continuously improving while learning.
