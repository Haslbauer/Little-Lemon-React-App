# Little Lemon React App

A React-based restaurant website built with reusable components and basic state management.

This project was created as part of the Meta Front-End Developer course and helped me strengthen my understanding of React fundamentals in a practical way. The focus was on component structure, props, state, and building a responsive user interface.

## Features
- Responsive restaurant website layout
- Reusable React components
- Navigation bar with section links
- Menu and specials sections
- Table booking form
- Basic filtering and state-driven UI elements

## Tech Stack
- React
- JavaScript
- CSS
- Vite

## What I Learned
- Building applications with reusable React components
- Passing data through props
- Managing UI state with `useState`
- Structuring a small React project
- Creating responsive layouts for different screen sizes
- Organizing project files and components in a clear way
